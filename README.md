# Education Online Website

## 🔍 Overview
Academia is a modern online education platform designed to provide high-quality learning experiences to students worldwide. The platform offers a wide range of courses across various subjects, including technology, business, arts, and personal development.
---

## 🛠️ Technologies Used

- **React.js** as the React framework
- **React Router** for navigate between pages-
- **React reveal** ⚠️ Note: This is an older project built with libraries common at the time.

---


### To Install and Run the Project

1. Clone the repository:
    ```bash
   https://github.com/Lotfy12/Education-Online.git
    ```
2. Navigate to the project directory:
    ```bash
    cd project-directory
    ```
3. Install the dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```
4. Run the development server:
    ```bash
    npm run dev
    # or
    yarn dev
    ```
